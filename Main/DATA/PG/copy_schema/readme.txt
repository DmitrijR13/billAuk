Скрипты для создания новой копии схемы (без данных).
Перед запуском необходимо настроить параметры подключения к серверу БД.
Для этого необходимот отредактировать переменные в файле copy_schema_structure:
  tmp_file - путь и имя временного файла, в который будет выгружена структура
  host - адрес сервера
  port - порт сервере
  user - имя пользователя
  dbname - имя базы данных
  PGPASSWORD - пароль пользователя

Можно удалить параметр PGPASSWORD из файла, тогда пароль будет запрашиваться при выполнении команд.

Запуск (для Windows):
> copy_schema_structure.cmd старое_имя_схемы новое_имя_схемы

Запуск (для nix):
> copy_schema_structure.sh старое_имя_схемы новое_имя_схемы
